The default content surface — crisp white on warm paper, hairline border, soft shadow.

```jsx
<Card>Default</Card>
<Card variant="raised" interactive>Clickable, lifts on hover</Card>
<Card variant="ink">Dark block for contrast / success moments</Card>
```

- `variant`: `default` · `raised` · `flat` · `outline` · `ink`
- `interactive` adds the hover-lift; use for tappable list items / job cards.
