KPI tile for the Company/Agency dashboard. Value is set in Roboto Mono.

```jsx
<StatTile label="Paid unlocks this week" value="1,284" icon="lock-key-open" delta="+12% vs last" deltaDir="up" />
<StatTile label="Repeat-unlock rate" value="62%" icon="repeat" delta="hero health metric" deltaDir="flat" />
```

- North-star metric is *weekly paid unlocks*; health metric is *repeat-unlock rate*.
