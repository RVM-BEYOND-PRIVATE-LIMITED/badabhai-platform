# Product Bible (placeholder)

The long-form product/strategy "bible" lives here as it is written. For now, the
locked, authoritative summary is:

- **Product:** BadaBhai — an AI placement-team product for blue/grey-collar India,
  starting with industrial manufacturing (CNC/VMC).
- **Phase 1:** Worker Profiling + Profile Generation only.
- **Principles:** event-first; chat-first onboarding; LLMs assist (profiling,
  canonicalization, explanations) but **never rank/reject/decide matches**;
  pseudonymization before every LLM call; DPDP consent + worker protection are
  launch gates.
